# Sistema de Facturación ALQUIPC

Aplicativo web desarrollado en HTML, CSS y JavaScript para la empresa ALQUIPC.

## Funcionalidades

- Registro de número de equipos (mínimo 2)
- Registro de días iniciales y días adicionales
- Cálculo automático del valor total
- Incremento del 5% si es fuera de la ciudad
- Descuento del 5% si es dentro del establecimiento
- Descuento del 2% por día adicional (máximo 10 días para evitar pérdidas)
- Generación automática de ID Cliente
- No permite impresión (solo visualización digital)

## Cómo usar

1. Abrir el archivo index.html en el navegador.
2. Ingresar los datos solicitados.
3. Presionar "Generar Factura".
4. Se mostrará el resumen con el valor total.

## Despliegue en GitHub Pages

1. Subir el proyecto a un repositorio en GitHub.
2. Ir a Settings > Pages.
3. Seleccionar rama main.
4. Guardar y usar el enlace generado.
